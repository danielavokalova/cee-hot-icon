# CEE Hot Icon

GitHub Pages site: https://danielavokalova.github.io/cee-hot-icon/

## Purpose

Simple download page for CEE Admin Console desktop/taskbar badge installers.

## Active Download Links

- Windows: `downloads/windows/cee-badge-windows-v14.zip`
- Linux: `downloads/linux/cee-badge-linux-v3.zip`

## Notes

- Header logo uses `icons/cee-white-logo-clean.png`.
- Buttons are equal width and labels stay on one line.
- Windows installer creates `CEE Admin Console.lnk` with custom icon.
- Linux package contains `.desktop` launcher, icon, and install script.
